WhiteSpawn — плагин для Rust (Oxide/uMod), предоставляющий систему безопасного спавна с защищённой зоной, таймером телепортации, проверкой враждебности и автоматическим управлением респавном. Разработчик — **whitecristafer**, при спонсорстве **infunv.ru** для проекта **evolve.infunv.ru**.

## Обзор

WhiteSpawn даёт администраторам сервера полноценное решение для спавна:

- Телепорт в одну кнопку в настраиваемую точку
- Настраиваемая задержка с отменой при враждебности
- Защищённая зона вокруг спавна (неуязвимость, запрет строительства и урона)
- Автоматическая телепортация при первом заходе и после смерти
- Приветственные и респавн‑сообщения
- Опциональная команда `/respawn` (потеря вещей)
- Обходы таймера и принудительного спавна по правам

Плагин автоматически определяет Outpost или Bandit Town как спавн по умолчанию, если точка не установлена вручную.

## Возможности

- **Телепорт на спавн**: `/spawn` перемещает игрока в установленную точку
- **Настраиваемый таймер**: задержка перед телепортацией (по умолчанию 5 секунд)
- **Проверка враждебности**: телепорт отменяется, если игрок становится враждебным во время отсчёта
- **Безопасная зона**: сфера вокруг спавна, в которой игроки неуязвимы, строительство и урон заблокированы (кроме админов)
- **Авто‑спавн при первом заходе**: новые игроки телепортируются на спавн с приветственным сообщением
- **Авто‑спавн после смерти**: игроки возвращаются на спавн после респавна с сообщением
- **Обход админа**: `whitespawn.admin` позволяет строить, наносить урон и игнорировать ограничения в зоне
- **Обход таймера**: `whitespawn.bypasstimer` убирает задержку телепортации
- **Обход спавна**: `whitespawn.bypassspawn` отключает автоматическую телепортацию при первом заходе и после смерти
- **Команда респавна**: `/respawn` убивает игрока и возрождает на спавне (вещи теряются) – можно отключить
- **Авто‑определение**: если спавн не установлен, плагин ищет Outpost или Bandit Town при старте
- **Настройка радиуса**: радиус зоны можно менять через команду или конфиг
- **Локализация**: встроенная поддержка английского и русского языков
- **Сообщения в чате**: настраиваются через lang‑файлы, с поддержкой аватарки по SteamID

## Команды

| Команда | Описание | Право |
| --- | --- | --- |
| `/spawn` | Телепорт на точку спавна | Нет (если включено) |
| `/setspawn` | Установить точку спавна на текущей позиции | `whitespawn.admin` |
| `/ws radius <число>` | Установить радиус зоны (в метрах) | `whitespawn.admin` |
| `/ws status` | Показать статус плагина (включён, радиус, таймер) | Нет |
| `/ws help` | Показать справку | Нет |
| `/respawn` | Убить себя и респавнуться на спавне (потеря вещей) | Нет (если включено) |

## Права

| Право | Описание |
| --- | --- |
| `whitespawn.admin` | Полный доступ: установка спавна, смена радиуса, обход всех ограничений (строительство, урон, таймер, спавн) в зоне |
| `whitespawn.bypasstimer` | Убирает задержку телепортации – мгновенный телепорт |
| `whitespawn.bypassspawn` | Отключает автоматическую телепортацию при первом заходе и после смерти (игрок спавнится обычно) |

## Конфигурация

WhiteSpawn автоматически создаёт конфигурационный файл при первом запуске.  
Основные настройки (в `oxide/config/WhiteSpawn.json`):

```json
{
  "Settings": {
    "Enabled": true,
    "SpawnTimer": 5.0,
    "Radius": 10.0,
    "WelcomeMessageEnabled": true,
    "RespawnMessageEnabled": true,
    "RespawnCommandEnabled": true,
    "FindOutpostFirst": true
  }
}
```

| Поле | Описание |
| --- | --- |
| `Enabled` | Включить/отключить плагин |
| `SpawnTimer` | Задержка перед телепортом в секундах (0 = мгновенно) |
| `Radius` | Радиус безопасной зоны вокруг спавна (в метрах) |
| `WelcomeMessageEnabled` | Показывать приветственное сообщение при первом заходе |
| `RespawnMessageEnabled` | Показывать сообщение о респавне после смерти |
| `RespawnCommandEnabled` | Включить/отключить команду `/respawn` |
| `FindOutpostFirst` | Если true, искать Outpost как спавн по умолчанию; если false – Bandit Town |

## Как это работает

1. **Телепорт на спавн**:  
   При использовании `/spawn` плагин проверяет враждебность. Если игрок не враждебен, запускается таймер (если не используется обход). По окончании таймера игрок телепортируется на спавн со звуковыми эффектами. Если игрок становится враждебным во время отсчёта, телепорт отменяется.

2. **Безопасная зона**:  
   Все игроки внутри радиуса вокруг спавна неуязвимы. Строительство и урон по постройкам заблокированы. Администраторы с правом `whitespawn.admin` могут игнорировать все эти ограничения.

3. **Первый заход**:  
   Новые игроки автоматически телепортируются на спавн через 0.5 секунды (если у них нет права `whitespawn.bypassspawn`). Показывается приветственное сообщение.

4. **После смерти**:  
   При респавне игроки возвращаются на спавн (если нет права обхода). Показывается сообщение о респавне.

5. **Команда респавна**:  
   `/respawn` очищает инвентарь, убивает игрока и принудительно возрождает. Затем игрок телепортируется на спавн. Команду можно отключить в конфиге.

6. **Спавн по умолчанию**:  
   Если спавн не установлен вручную, плагин ищет Outpost (или Bandit Town) при старте сервера и устанавливает точку на его позиции.

## Система обновлений

WhiteSpawn **не** содержит автоматическую проверку обновлений. Поскольку плагин поддерживается whitecristafer, обновления выходят вручную. Администраторам следует периодически проверять репозиторий на наличие новых версий.

## Установка

1. Скачайте `WhiteSpawn.cs` и поместите в папку `oxide/plugins`.
2. Перезапустите сервер или выполните `oxide.reload WhiteSpawn`.
3. Плагин создаст конфигурацию и файлы данных по умолчанию.
4. Настройте `oxide/config/WhiteSpawn.json` под свои нужды.
5. Выдайте права администраторам (например, `oxide.grant group admin whitespawn.admin`).
6. Установите точку спавна через `/setspawn` (или позвольте плагину автоопределить Outpost/Bandit).

## Локализация

Встроенные языки:
- Английский (по умолчанию)
- Русский

Вы можете добавить или изменить переводы, отредактировав `lang/en.json` и `lang/ru.json` в папке `oxide/lang`. Плагин использует те же ключи сообщений, что определены в методе `LoadDefaultMessages`.

## Логирование

WhiteSpawn выводит в консоль важные события:
- Установка спавна (с именем игрока и позицией)
- Изменение радиуса
- Попытки телепортации (заблокированы/отменены)
- Стартовый баннер с информацией о плагине

Все сообщения помечаются префиксом `[WhiteSpawn]` для удобного поиска.

## Требования

- Выделенный сервер Rust
- Oxide/uMod (рекомендуется последняя версия)
- Поддержка C# плагинов

## Примечания

- Звуки телепортации используют встроенный префаб телепорта.
- Безопасная зона центрирована ровно в точке, установленной `/setspawn`.
- Проверка враждебности использует `player.IsHostile()` (доступно в актуальных версиях Rust).
- Данные спавна хранятся в `oxide/data/WhiteSpawn/spawn.json`.
- Команда `/respawn` удаляет все вещи – игрок респавнится с базовым набором (камень, факел и т.д.).
